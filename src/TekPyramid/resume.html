To avoid wheel theft, there are alarming systems in existence to alert the owner or nearby people that wheel theft is going to happen but this work features an antitheft system for car wheels and rims by using the Linear Variable Differential locking system between the wheel rim and the brake rotor disk. This work features the anti-theft system for car wheels and rims by using the Linear Variable Differential transformer locking system between the wheel rim and the brake rotor disk.
